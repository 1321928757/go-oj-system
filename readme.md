# go-web-starter

`go-web-starter` 是一个基于form 和 `gin` 框架的脚手架，方便用户快速的编写curd操作。



## 目录结构

```
project
├───app
│   ├───common
│   │   ├───request
│   │   └───response
│   ├───controller
│   ├───dao
│   ├───middleware
│   └───service
├───bootstrap
├───config
│   └───fileDriver
├───docs
├───global
├───model
├───routes
├───static
├───storage
│   ├───app
│   │   └───public
│   │       └───local
│   │           └───test
│   └───logs
├───test
└───utils
    └───storage
        ├───cos
        ├───kodo
        ├───local
        └───oss
```



## 安装和使用

### 安装

```
go mod tidy
```



### 运行

```
go run main.go
```
